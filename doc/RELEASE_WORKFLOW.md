## 版本管理流程

```
┌─────────────────────────────────────────────────────────────────┐
│                     版本号管理                                    │
└─────────────────────────────────────────────────────────────────┘

                    开发阶段
                       │
                       ▼
              ┌────────────────┐
              │ v0.1.0-alpha.1 │  Alpha 测试
              └────────┬───────┘
                       │
                       ▼
              ┌────────────────┐
              │ v0.1.0-beta.1  │  Beta 测试
              └────────┬───────┘
                       │
                       ▼
              ┌────────────────┐
              │ v0.1.0-rc.1    │  Release Candidate
              └────────┬───────┘
                       │
                       ▼
              ┌────────────────┐
              │ v0.1.0         │  首个发布版本
              └────────┬───────┘
                       │
        ┌──────────────┼──────────────┐
        │              │              │
        ▼              ▼              ▼
┌───────────┐  ┌──────────────┐  ┌──────────┐
│ Bug 修复  │  │ 新增功能     │  │ 重大变更 │
└─────┬─────┘  └──────┬───────┘  └────┬─────┘
      │               │               │
      ▼               ▼               ▼
┌───────────┐  ┌──────────────┐  ┌──────────┐
│ v0.1.1    │  │ v0.2.0       │  │ v1.0.0   │
│ PATCH     │  │ MINOR        │  │ MAJOR    │
└───────────┘  └──────────────┘  └──────────┘

语义化版本: vMAJOR.MINOR.PATCH

MAJOR: 不兼容的 API 变更
MINOR: 向后兼容的功能新增
PATCH: 向后兼容的问题修复
```

## 快速参考

### 发布命令速查

```bash
# 查看帮助
make help

# 测试发布配置
make release-test

# 创建 tag
make tag

# 发布
make release

# 查看所有 tags
git tag -l

# 删除 tag
git tag -d v1.0.0
git push --delete origin v1.0.0
```

### 用户安装命令速查

```bash
# 安装脚本
curl -sSL https://raw.githubusercontent.com/goupter/goupter/main/install.sh | bash

# go install
go install github.com/goupter/goupter/cmd/goupter@latest

# Homebrew
brew tap goupter/tap
brew install goupter

# 验证
goupter --version
```
